System Requirements

This website has been tested and verified to work under the following operating systems, web browsers, and Java versions. Other versions may work, but are not guaranteed to do so.

Most problems users experience are caused by out of date versions of Java.

Operating Systems

  • Microsoft Windows XP and later
  • Apple OS X 10.5.8 and later
  • Ubuntu Linux 10.04 and later

Web Browsers

  • Mozilla Firefox 3.5 and later
  • Apple Safari 5 and later
  • Google Chrome 9 and later
  • Opera 11 and later
  • Microsoft Internet Explorer 8 and later

Java Versions

  • Java Version 6 and later
